La programación Frontend es el proceso de creación de la interfaz de usuario de un sitio web o aplicación. Se refiere a todo lo que un usuario ve y con lo que interactúa en una página web, como los botones, los menús, los formularios y el diseño visual en general. Los programadores Frontend utilizan lenguajes de programación como HTML, CSS y JavaScript para crear estas interfaces de usuario. En resumen, la programación Frontend es esencial para crear una experiencia de usuario atractiva y funcional en un sitio web o aplicación.

Etiqueta <p>

La etiqueta <p> (párrafo) por defecto al usarse tiene un ancho completo, osea que se adapta al ancho completo del navegador, por lo contrario el alto por defecto sin ningun caracter dentro será de 0px. Al poner un caracter dentro de esta etiqueta el alto de este por defecto será de 18px.

Etiqueta <span> y <h1>

La etiqueta “span” y “a” por defecto tiene un alto y ancho automático, ya que estos se ajustan al alto y ancho del contenido que contiene.

Etiqueta <div>

La etiqueta div por defecto ocupa todo el ancho disponible en el navegador y un alto de 0px, al introducir texto dentro de el, el alto aumenta a 18px.

Todo esto puede variar dependiendo del navegador, cabe decir que cada uno de estos parametros pueden ser modificados desde el CSS, teniendo la posibilidad de fijar un acho y alto .

Ejemplo tabla HTML

Aquí podras encontrar un ejemplo básico de como utilizar las tablas de HTML, para mostrar los datos que estimes convenientes. Podrás encontrar el código fuente despues de la imagen.

Lo que nos permiten hacer las tablas es desplegar nuestra información en forma de filas y columnas. Los antiguos desarrolladores utilizaban esta etiqueta para maquetar nuestros sitios web, en estos momentos esto ya quedo obsoleto, ya que se crearon nuevas etiquetas especialmente para maquetar nuestros sitios.

Para declar nuestra tabla se utiliza la etiqueta <table>, obviamente esta se debe cerrar como es debido </table> (etiqueta que abre, etiqueta que cierra)

  • <tr> - Etiqueta para declarar filas.

    • Dentro de las filas existen columnas, la primera debe ser la columna de nuestros encabezados, luego de eso las columnas normales

  • <th> para declarar columnas de encabezados.
  • <td> para declarar columnas normales y habituales.

    • Interpretación tabla HTML

    Código HTML (table.html):

    Podemos utilizar el atributo border para mostrar un relleno en la tabla.

    Utilizando CSS podriamos darle más estilo a las tablas, cambiando hasta su tipo de borde y modificando colores. 

                    
        <table border="fill">
    
            <!-- La siguiente fila es para declarar mis encabezados:  -->
