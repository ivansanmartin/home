Programación Frontend

La programación Frontend es el proceso de creación de la interfaz de usuario de un sitio web o aplicación. Se refiere a todo lo que un usuario ve y con lo que interactúa en una página web, como los botones, los menús, los formularios y el diseño visual en general. Los programadores Frontend utilizan lenguajes de programación como HTML, CSS y JavaScript para crear estas interfaces de usuario. En resumen, la programación Frontend es esencial para crear una experiencia de usuario atractiva y funcional en un sitio web o aplicación.

Etiqueta <p>

La etiqueta <p> (párrafo) por defecto al usarse tiene un ancho completo, osea que se adapta al ancho completo del navegador, por lo contrario el alto por defecto sin ningun caracter dentro será de 0px. Al poner un caracter dentro de esta etiqueta el alto de este por defecto será de 18px.

Etiqueta <span> y <h1>

La etiqueta “span” y “a” por defecto tiene un alto y ancho automático, ya que estos se ajustan al alto y ancho del contenido que contiene.

Etiqueta <div>

La etiqueta div por defecto ocupa todo el ancho disponible en el navegador y un alto de 0px, al introducir texto dentro de el, el alto aumenta a 18px.

Todo esto puede variar dependiendo del navegador, cabe decir que cada uno de estos parametros pueden ser modificados desde el CSS, teniendo la posibilidad de fijar un acho y alto que estimes conveniente.
